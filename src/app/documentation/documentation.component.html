Get necessary modules to run the app.

App is connected to the database. Server/backend is running concurrently with angular [yarn serve & ng serve]. ng serve is conneced to yarn serve

Documentation

Employee App has multiple components to build entire app. Each component is given meaningful names to understand easily for new users. The app hierarchy:

  • App
    • Header
      • Who We Are
      • News
      • Documentation
      • Sign In / Up
    • Main Page
      • Employees List
      • Employees Detail
    • Footer
      • Social media
      • Contact Form
    • Data
      • Data Retrieve
      • Filter Pipe

Header

Contains the home page icon, relative links and search icon If youclick the search icon, you can search thru employees by first name or last name. Home icon returns to home page, else is visiting other pages.

Main Page

Left side of the page is a nav bar that you can scroll down for all employees or you can filter it by using A-J K-Z filter. You clikc employee name / last name to see employee details on the right side of the page.

Footer

Contains our social media links and a contact for to send email to us.

Data

'Data.service.ts' file contains url. This will be static. Data will read from excel file Contains data service and Filter pipe. Data service used for dump all the information from xlsx file and create an Employee object. Data Service is also sorting the Employees by first name for all of them or partially. Pipe Filter is a transform to search employee by first / last name.

Database

'Data.service.ts' file also contains databseUrl. This will be dynamic. Data will read from excel database.
